[Github Blog] 마크다운(Markdown) 작성법
===================================

### 1. Header

* 문서의 제목(큰제목)
  
  ```
  Main title
  ===========
  ```
  
  Main title
  ============
  
* 문서의 부제목(작은제목)
  
   ```
   sub title
   ---------------
   ```
   
   sub title
   ------------
   
* 글머리 : H1~H6

  ```
  # H1
  ## H2
  ### H3
  #### H4
  ##### H5
  ###### H6
  ```
  
  # H1
  ## H2
  ### H3
  #### H4
  ##### H5
  ###### H6
